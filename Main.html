Introduction

Python is a popular programming language that is used for web development (server-side), software development, mathematics and system scripting. Python works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc), It has simple syntax that is similar to the English language and developers may need to write fewer lines of code than some other programming languages. It runs on an interpreter system, meaning that code can be executed as soon as it is written. This means that prototyping can be very quick and can be treated in multiple ways.


Python can be:

  • Used on a server to create web applications.
  • Used alongside software to create workflows.
  • Connect to database systems. It can also read and modify files.
  • Used to handle big data and perform complex mathematics.
  • Used for rapid prototyping, or for production-ready software development.

  • General understanding of programming and its languages.

Hello World

To get started writing with Python, create a file with any name and with the extension of .py and open it up using any text editor or IDE (Integrated Development Environment). After creating the file, write your first "Hello World" code like this:

print("Hello World")

Once you are done, hit F5 or run on the menu bar to execute your code!

Variables

Variables are containers in which you can store some values that can be used later in your code, variables are very useful for storing values to be used later repeatedly and for mathematical operations as well.


Python variables do not need to be declared with any particular type, you can change the type of the variable after declaring it any number of times. There are multiple data types in Python like: String (used for text), Integer (used for number) and Bool (used for true or false).

Declaring variables

You can declare a variable like this:

x = 10

You can change the value of the variable in the same way as declaring a variable.

Loops

A "for" loop in Python is the most basic loop that is used in Python. It is used for iterating over a sequence a number of times that can be defined by the programmer. In a "for" loop, the loop is assigned a variable in which it adds or subtracts a certain number (usually 1) and keeps track of the number, after the number reaches its limit that is defined by the programmer, the loop exits itself.


You can create a "for" loop like this:

for x in range(25):
    print(x)

In the above code, x is a variable (doesn't need to be declared before the loop) in which 1 is added every time it goes through the loop, once it reaches 25, it exits the loop. print(x) prints the value of x every time it goes through the loop, so the output would be like this: 0, 1, 2, 3 and so on.


A "while" loop, iterates over a sequence as long as a condition is true. For example:

x = 0
while x < 5:
    x++
    print(x)

In the above code, x is a variable that is declared a value as 0, the "while" loop checks if x is under 5, if it is, it adds 1 to x as ++ adds 1 to the variable which is x in this case and it prints the value of x. After x is equal to or greater than 5, it exits the loop.

if...else statement

The if statement is used for executing (a) statement(s) if a logical operation is true. The optional else statement is used for executing (a) statement(s) if the logical condition in the if statement is not true.


An if...else statement can be written like this:

x = 10
if x > 5:
    print("x is greater than 5.")
else:
    print("x is not greater than 5.")
